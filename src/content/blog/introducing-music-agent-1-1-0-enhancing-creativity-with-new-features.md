I’m thrilled to announce the release of **Music Agent 1.1.0**!  
This version marks an exciting step forward in empowering creators to explore the intersection of music and technology. Whether you’re an experienced Sonic Pi enthusiast or just starting your journey in generative music, this update brings tools and improvements designed to elevate your creative experience.

## What’s New in MusicAgent 1.1.0?

### 1\. **Anthropic API Support**

Music Agent now supports the Anthropic API for language model interaction, offering an alternative to OpenAI. Note that Anthropic does not currently provide image generation, so album covers will not be generated when using the Anthropic API. To switch between OpenAI and Anthropic, users can set the API keys in their configuration file or as environment variables.

### 2\. **Expanded Agent Capabilities**

In this version, agent configurations are smarter and more versatile than ever. They can now:

-   Handle a broader range of agents .
-   Collaborate seamlessly to ensure harmony across all musical layers.

### 3\. **Model Recommendations**

We’ve added additional models for users to choose from, allowing greater customization and flexibility in creative workflows.

### 4\. **Improved Sonic Pi Code Integration**

Coding music in Sonic Pi is at the heart of Music Agent. Version 1.1.0 introduces:

-   Better code review suggestions from agents.
-   Enhanced support for debugging and optimizing Sonic Pi scripts.
-   New sample patterns to jumpstart your compositions.

### 5\. **User Interface Enhancements**

The CLI interface has been streamlined for clarity and efficiency, making it even easier to launch agents and manage your music projects.

### 6\. **Enhanced Documentation**

The Music Agent user manual has been expanded to include a new section on configuring and running the agent with the Anthropic API (e.g., RUST installation will be required). This section also clarifies the limitations of using Anthropic, such as the lack of album cover generation.

### 7\. **Performance Upgrades**

We’ve improved the underlying architecture, ensuring faster responses and smoother agent collaboration. This means less waiting and more creating.

## Why Upgrade?

Music Agent 1.1.0 isn’t just an update; it’s an invitation to reimagine how you compose and collaborate. By leveraging the enhanced features, you can:

-   Save time while crafting complex musical pieces.
-   Push the boundaries of generative music.
-   Collaborate effectively, whether with other users or AI-driven agents.
