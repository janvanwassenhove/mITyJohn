**Hold onto your MIDI cables, because Music Agent v1.4.0 is here, and it’s bringing the rhythm!**

This release isn’t just an update—it’s a full-blown remix. We’ve revamped the frontend, supercharged playback, and fine-tuned agent interactions so that composing with AI feels smoother than ever. Let’s dive into the beats and bytes of what’s new.

## The Big Changes

### A Fresh Vue (Literally)

-   We’ve upgraded to Vue.js 3, making the interface faster, sleeker, and ready for the future.
-   A shiny new navigation bar keeps everything within easy reach.

![](/wp-content/uploads/2025/03/ma_webapp-768x638.webp)

### Smarter Browsing & Song Management

-   **Songs & Samples Listing** – Easily browse tracks, see covers, and peek at the Sonic Pi code.
-   **Playback Upgrades** – Listen to samples directly in the interface without hassle.
-   **Search & Pagination** – No more endless scrolling—find what you need instantly.

![](/wp-content/uploads/2025/03/songlibrary-1024x912.webp)

### AI Gets a Memory

-   **Timeline View** – See how the agent has been evolving in one streamlined view.
-   **Conversation History** – Keep track of past interactions and never lose an idea.
-   **Detailed Agent Insights** – View parameters, Sonic Pi code, and logs at a glance.

### Customization & Visual Enhancements

-   Adjust styling and Sonic Pi settings straight from the web app.

![](/wp-content/uploads/2025/03/settings_sonic_pi-1024x594.webp)

![](/wp-content/uploads/2025/03/settings_general-1024x538.webp)

-   A **spectrum analyzer** now gives a visual touch to your sample playback.

![](/wp-content/uploads/2025/03/samples_playback-1024x1013.webp)

### Smarter Metadata & Automation

-   Launch metadata listing right from the interface, with a progress bar keeping you in the loop.

## Under-the-Hood Fixes & Optimizations

### Sound & Playback Tweaks

-   Fixed playback hiccups and improved handling of sharp notes.
-   Sample paths now load correctly—because AI shouldn’t make you hunt for files.

### Code & Structural Upgrades

-   Python scripts now live in the `bce` folder for better organization.
-   The directory structure got a major cleanup, making future updates smoother.
-   API communication with Sonic Pi is now more robust and reliable.

### UI & Documentation Improvements

-   Scrollbar styling fixes in the song library for a cleaner experience.
-   Improved layout for the sample page.
-   Updated README files to help you get started faster.
-   New **integration test for Sonic Pi** to ensure everything runs like a dream.

## Ready to Jam?

This is the most refined version of Music Agent yet, and it’s ready to transform your AI-assisted music creation.

Check it out now: [Music Agent v1.4.0](https://github.com/janvanwassenhove/MusicAgent/releases/tag/v1.4.0)!

Get ready to compose, experiment, and let the AI take your music to the next level—without missing a beat!
