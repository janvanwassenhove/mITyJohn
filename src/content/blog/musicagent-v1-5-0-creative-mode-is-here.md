The latest MusicAgent release is here — version 1.5.0 introduces **Creative Mode**: a new visual way to compose music with Sonic Pi, while chatting live with your AI music assistant.

![](/wp-content/uploads/2025/04/CreativeMode-1024x502.png)

Creative Mode – Starting from scratch

If you prefer composing with visuals instead of just code — or simply like seeing your song’s structure take shape in real time — this update’s for you.

---

## From Code to Creation

Until now, MusicAgent focused on generating Sonic Pi code through a Multi Agent System. That still works (and gets smarter), but **Creative Mode** adds a whole new layer on top. You get:

-   a **visual timeline** to see what plays when
-   **real-time code sync** while composing together
-   easy **multi-sample selection and layering**
-   and most importantly: a more dynamic experience where you and MusicAgent co-create, tweak, and refine ideas on the fly

It’s like jamming in a studio, except one of the bandmates speaks fluent Ruby.

![](/wp-content/uploads/2025/04/SonicPi_Visualisation.png)

Sonic Pi Code Song Visualization

## Sample Everything

The creative mode also includes a new **multi-sample selector**, supporting both internal and external files (like `.wav`). Handy for layering textures, building punchy drops, or experimenting with surprise elements.

Samples drop directly into the timeline and are integrated into the live Sonic Pi loop — with effects or structure hints if needed.
